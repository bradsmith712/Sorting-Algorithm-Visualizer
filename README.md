## Sorting Algorithm Visualizer

Tool for visualizing common sorting algorithms. Launch tool here - https://bradsmith712.github.io/Sorting-Algorithm-Visualizer/

### Algorithms Included

* Bubble Sort
* Insert Sort
* Selection Sort
* Quick Sort

#### Algorithms In Progress

* Heap Sort
* Merge Sort

## Tech Stack

* React (+ hooks)
* Redux
* Typescript
